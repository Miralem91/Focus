ReadMe: 

Opcije koje ima korisnik webstranice: 
  -Registracija korisnika,
  -Loggin i loggout korisnika
  -Citanje vijesti
  -Listanje kategorija
  -Citanje vijesti prema kategorijama
  -Dodavanje komentara na vijesti
  

Opcije koje ima admin webstranice:

 -Menađment korisnika što obuhvata : dodavanje, brisanje i izmjenu podataka korisnika kao što su username, password i uloga (da li je korisnik samo user ili je admin)
 -Menađment vijesti što obuhvata : kreiranje vijesti, editovanje vijesti i brisanje vijesti
 -Menađment kategorija što podrazumjeva : dodavanje kategorija
 -Menađment komentara što podazumjeva : dodavanje komentara, editovanja komentara drugih korisnika  te brisanje komentara
 
Kreirana je posebna stranica za ADMIN PAGE (admin.php) na kojoj su dostupne sve navedene opcije. Obični korisnik ne može pristupiti stranici admin.php.

Sav sadržaj se prikazuje dinamički skidanjem sa baze podataka osim dvije stranice about.php i buyit.php koje su hardcodirane. 

Prikaz mnogih stranica se razlikuje obzirom koja je uloga ulogovanog korisnika, da li je admin ili samo user.

Na CSS stilizaciju nisamo obratio preveliku pažnju, bilo mi je bitno da sve funkcioniše kako treba..izgled web stranice se može polirati po potrebi.
